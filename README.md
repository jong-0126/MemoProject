## 📌 Memo API
### 📖 개요

Memo API는 메모를 생성, 조회, 수정 및 삭제할 수 있는 RESTful API입니다. 모든 데이터는 JSON 형식으로 주고받으며, 각각의 메모는 다음과 같이 구성됩니다:
- 식별자 (id)
- 제목 (title)
- 내용 (contents)

## 📌 API 명세
### 1️⃣ 메모 생성 (CREATE)
- 메모 생성 시 제목, 내용이 필요하다.
- 생성된 데이터(식별자, 제목, 내용)가 응답된다.

### 2️⃣ 메모 전체 목록 조회 (READ)
- 여러 개의 데이터를 배열 형태로 한번에 응답한다.
- 데이터가 없는 경우 비어있는 배열 형태로 응답한다.

### 3️⃣ 특정 메모 조회 (READ)
- 조회할 memo에 대한 식별자 id값이 필요하다.
- 조회된 데이터가 응답된다.
- 조회될 데이터가 없는 경우 Exception이 발생한다.
  
### 4️⃣ 메모 전체 수정 (UPDATE - PUT)
- 수정할 memo에 대한 식별자 id값이 필요하다.
- 수정할 요청 데이터(제목, 내용)가 꼭 필요하다.
- 수정된 데이터가 응답된다.
- 수정될 데이터가 없는 경우 Exception이 발생한다.

### 5️⃣ 메모 제목 일부 수정 (UPDATE - PATCH)
- 수정할 memo에 대한 식별자 id값이 필요하다.
- 수정할 요청 데이터(제목)가 꼭 필요하다.
- 수정된 데이터가 응답된다.
- 수정될 데이터가 없는 경우 Exception이 발생한다.

### 6️⃣ 메모 삭제 (DELETE)
- 삭제할 memo에 대한 식별자 id값이 필요하다.
- 삭제될 데이터가 없는 경우 Exception이 발생한다.
